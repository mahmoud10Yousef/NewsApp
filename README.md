# NewsApp 
 
 News app provide  information about news , show news details and do a search on news .
 
# Built Using 
 -  Swift 5
 -  MVVM 
 -  RXSwift 
 -  Cocapods (Kingfisher , Almofire , progressHUD) 
 -  Restful API
